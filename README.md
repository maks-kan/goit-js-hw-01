# goit-js-hw-01

**Автор:** Максим Канюка

## Опис

Домашнє завдання №1 з курсу JavaScript від GoIT.

## Завдання

- **task-1.js** — Замовлення дроїдів: функція `makeTransaction(quantity, pricePerDroid)`, що повертає повідомлення про загальну вартість замовлення.
- **task-2.js** — Доставка товару: функція `getShippingMessage(country, price, deliveryFee)`, що повертає повідомлення про вартість доставки.
- **task-3.js** — Ширина елемента: функція `getElementWidth(content, padding, border)`, що обчислює загальну ширину елемента з урахуванням моделі `border-box`.
